# FXIJコンフィグツール 操作ヘルプ

## 概要

FXIJコンフィグツールは、Mistralシステムの各種サーバー設定やシステム操作をブラウザから直感的に行うための管理用GUIアプリケーションです。
左側のサイドバーメニューから必要な機能を選択して操作します。

---

## 各機能の説明

### 1. サーバー情報
システムの現在の状態を確認する画面です。
- **CPU / メモリ / ディスク**: ハードウェアの利用状況や構成情報が表示されます。
- **バージョン情報**: Mistralバージョン、PDCバージョン、HIFバージョンの各種ファームウェアやソフトウェアのバージョンが一覧で表示されます。

### 2. サーバー設定
Mistralシステムを構成するパラメーターを変更する画面です。
- **現在の設定**: `mistral.json` などの設定ファイルに保存されている、現在のパラメータ値が表示されます。
- **ヘッド構成**: ご利用の装置に合わせたヘッド構成 (Type500/Type1000など) を選択します。
- **ヘッド印刷向き**: 印刷の出力方向（正方向・逆方向）を設定します。
- **サーバーIPアドレス**: 構成する各サーバー（サーバー1〜4）のIPアドレスを指定します。
- これらを変更後、「**設定を更新**」ボタンを押すことで設定ファイルへ反映されます。
- 「**設定をインポート**」から、`.tgz` 形式のパッケージを使って設定を一括でインポートすることも可能です。

### 3. サーバー時刻設定
- サーバーのシステム日時がずれている場合などに、本ツールを開いている**「ブラウザのPCの日時」にサーバー側の日時を合わせる（同期する）**機能です。
- 「**サーバー時刻更新**」ボタンをクリックするだけで、コマンドが発行され自動で適用されます。

### 4. SSD RAID設定
マウントや、システムのRAIDアレイ構成の初期化を行います。
- 現在の「OSディスク」と「RAIDディスク」の状態が画面上部に分けて表示されます（OSディスクはRAID構成の対象外として安全に保護されます）。
- 「**マウント実行**」 / 「**マウント解除**」: RAIDボリューム（`/dev/md127` 等）をシステムにマウント・アンマウントします。
- 「**RAID初期化**」: データ用ディスクの内容をすべて消去し、新しいRAIDアレイを構築します。※安全のため、OSディスク以外のディスクが「3台」揃っている場合のみ実行可能です。**データが完全に失われる**ため、実行前には必ず確認してください。

### 5. ログ取得
Mistralシステムが出力するログファイル（`/var/mistral/log/mistlog*`）を、手元のPCにダウンロードする機能です。
- 画面を開くと内部で自動的にログのアーカイブ（ZIPファイル）が作成されます。
- 処理完了後、画面に表示される「**ダウンロード**」ボタンから取得できます。

### 6. ログ検索
システム内にあるシステムログファイル (`mistlog.log`) をブラウザ上で検索できる機能です。
- 検索したいキーワードを入力し「**検索**」を押します。
- スペース区切りで「AND検索」、`|` 区切りで「OR検索」に対応しています。
- 検索ワードを「空白」のまま検索を実行すると、ログ全体を表示します（ログサイズが10MBを超える場合は、最新の10MB分のみ表示します）。

### 7. 印刷ページメモリ調整
GRUBの設定を利用して、Mistralに割り当てるCMA（Contiguous Memory Allocator）サイズを変更する機能です。
- 現在のサーバーメモリのサイズに基づいて、推奨されるページメモリ容量（16GB以上なら20GB、16GB未満なら10GB）とのチェックを行います。
- 推奨値と一致していない場合は赤字で警告メッセージが表示されるため、「**印刷ページメモリ調整**」ボタンから設定を更新してください。
- 実行後はシステムへのGRUB更新が走るため、適宜再起動が必要になる場合があります。

### 8. ソフトウェアアップデート
- 提供されたアップデート用ファイルを選択し、「**アップデート**」を押すことで、システム内の規定ディレクトリへファイルをアップロードし適用準備を行います。

### 9. 再起動/シャットダウン
- サーバーマシンの再起動 (`reboot`) 、シャットダウン (`shutdown`) 、およびMistralモジュールの再起動を実行します。
- 各種ボタンを押すと確認ダイアログが表示され、「はい」を選択した場合のみ実行されます。
- ※コンフィグツールにログインするとMistralサービスは一旦停止されるため、作業終了後はこちらの画面から「**モジュール再起動**」を実行してください。

---

## 留意事項
- **サービスの自動停止について**: 本ツールにログインすると、設定変更中の競合を防ぐためMistralサービスに関連するプロセス（dcmd, sg3sdなど）が一旦停止されます。作業を終える際は、「再起動/シャットダウン」メニューから「モジュール再起動」を実行してください。

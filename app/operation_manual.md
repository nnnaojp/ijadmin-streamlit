# FXIJコンフィグツール 操作マニュアル

## 1. ログ取得 (Log Retrieval)

![ログ取得画面](./images/log_download_screen_1765789933924.png)

サーバー内のログファイルをアーカイブ（zip）してダウンロードする機能です。

### 操作手順

1.  サイドメニューから「**ログ取得**」を選択します。
2.  **対象を選択**します。
    - **すべてのログファイル**: `/var/mistral/log` ディレクトリ全体をアーカイブします。
    - **最新のログファイル**: `/var/mistral/log/mistlog.log` ファイルのみをアーカイブします。
3.  **sudoパスワード**を入力します。
    - デフォルトで `ijadmin` が入力されています。変更が必要な場合は書き換えてください。
    - ※アーカイブ作成にはルート権限が必要なため、sudoパスワードが必須です。
4.  「**アーカイブ作成**」ボタンをクリックします。
5.  処理が成功すると、「**ダウンロード**」ボタンが表示されます。
6.  「**ダウンロード**」ボタンをクリックして、zipファイルを保存してください。

### エラーが表示される場合
アーカイブ作成に失敗した場合、画面下部にエラー内容（標準出力・標準エラー出力）が表示されます。パスワード間違いや、対象ファイルが存在しない可能性があります。

## 2. 日時表示 (Date Display)

![日時表示画面](./images/date_display_screen_1765789962049.png)

サーバーの現在日時を表示する機能です。

### 操作手順

1.  サイドメニューから「**日時表示**」を選択します。
2.  「**現在日時**」欄に、サーバーの `date` コマンドの実行結果が表示されます。
3.  日時の右上に表示される **コピーボタン**（クリップボードアイコン）をクリックすると、日時テキストをコピーできます。
4.  最新の日時を取得したい場合は、再度サイドメニューの「日時表示」をクリックするか、ブラウザをリロードしてください。
